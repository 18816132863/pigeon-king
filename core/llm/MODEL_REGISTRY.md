# 模型注册表 & 决策引擎

## 设计理念

每次收到消息时，由 `ModelRouter` 自动评估任务特征，匹配最佳模型。
不需要用户手动切模型，也不需要每次调用 AI——用规则引擎做轻量级决策。

## 分类维度

| 维度 | 值 |
|------|-----|
| 任务类型 | 闲聊 / 编程 / 推理 / 知识检索 / 多模态 / 设备控制 / 任务编排 |
| 复杂度 | 低 / 中 / 高 |
| 延迟要求 | 实时 / 可等待 / 不限 |
| 成本敏感 | 省钱 / 平衡 / 不要钱 |
| 输出形式 | 文字 / 代码 / JSON / 图片 / 文件 |
| 上下文长度 | 短(<4K) / 中(4-16K) / 长(>16K) |
