# V10.6.0 Reality-Connected Personal OS Agent

## 本阶段推进方向

V10.6 在 V10.5 持续智能个人 OS 基础上，继续升级为“真实世界连接执行层”。

核心目标：

1. 所有真实动作先做事实状态解析，不能假装知道。
2. 所有外部动作先做环境能力探测，不能假装已连接。
3. 不确定时必须转入搜索、探测、确认，不能假成功。
4. 外部写入、设备动作、安装、发送、删除必须过授权意图网关。
5. 工具调用不是按名字乱选，而是按工具能力契约协商。
6. 真实执行 broker 默认 dry-run，没有确认不产生真实副作用。
7. 长期项目具备 milestone、memory graph、review。
8. 自升级要有 lineage、diff risk、snapshot 和回滚要求。
9. 审计可回放，失败能生成下次预防规则。

## 新形态

Reality-Connected Personal OS Agent
真实世界连接个人操作系统智能体

主链路：

目标
→ 事实状态解析
→ 环境能力探测
→ 不确定性管理
→ 项目自治
→ 工具能力协商
→ 授权意图网关
→ 真实执行 broker
→ 自升级治理
→ 审计回放学习

## 安全边界

- 没有确认就 dry-run
- 没有授权就不真实执行
- 不确定时不声称成功
- 高风险 scope 必须显式同意
- 外部能力必须按契约协商
- 自升级必须具备快照和回滚
