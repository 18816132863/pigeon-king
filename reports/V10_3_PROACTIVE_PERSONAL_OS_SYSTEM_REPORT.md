# V10.3.0 Proactive Self-Extending Personal OS Agent

## 这版一次性推进的方向

V10.3 在 V10.2 自主运行闭环基础上继续增强：

1. 主动任务中枢 Mission Control
2. 多角色内部评审：critic / verifier / executor
3. 结果验证、审计、回滚闭环
4. 外部连接器注册和权限范围管理
5. 技能/方案自动发现
6. 可信来源策略
7. 扩展生命周期管理
8. 沙箱晋升门
9. 个人反馈路由
10. 行为模式挖掘
11. 风险容忍度校准
12. 主动个人 OS 编排器

## 新形态

Self-Evolving Personal Operating Agent
→ Bounded Autonomous Runtime Loop
→ Proactive Self-Extending Personal OS Agent

也就是：
一句话目标进入后，系统能主动建 mission、评审计划、发现能力缺口、查找候选方案、管理外部连接器、生成验证与回滚、写回个人模型。

## 安全边界

- 未知外部代码不能直接安装
- 新能力只能 sandbox / experimental
- 高风险 scope 需要审批
- 外部连接器默认 planned，必须授权
- 所有执行必须有 rollback 和 audit
