# V10.5.0 Continuous Intelligence Personal OS Agent

## 本阶段推进方向

V10.5 在 V10.4 战略自治大脑基础上，继续升级为“持续智能操作系统层”。

核心目标：

1. 持续监控重点，而不是每次从零开始。
2. 自己判断什么值得注意，避免打扰用户。
3. 做情景模拟和反事实复核，先想后做。
4. 对知识来源做质量评分、时效衰减和精炼。
5. 外部连接前先做隐私最小化和脱敏。
6. 运维监督连接器健康、漂移和事件。
7. 能力具备候选→沙箱→实验→可信→核心的生命周期。
8. 元学习：从成功和纠正中优化下一轮策略。

## 新形态

Continuous Intelligence Personal OS Agent
持续智能个人操作系统智能体

主链路：

信号输入
→ 注意力排序
→ 预算分配
→ 上下文刷新计划
→ 主动事项提案
→ 情景模拟
→ 知识精炼
→ 隐私边界
→ 运维健康
→ 能力生命周期
→ 元学习
→ 下一轮主动任务

## 安全边界

- 没有真实授权不执行外部动作
- 外部 payload 必须最小化和脱敏
- external write/device action/money 默认 0
- 连接器 unhealthy 或漂移过大时开 incident
- 能力不能直接从 candidate 变 core
